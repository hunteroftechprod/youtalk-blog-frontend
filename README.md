# YouTalk Blog — Frontend Project

## 📌 О проекте

Учебный проект — адаптивная вёрстка блога психологического сервиса **YouTalk**.

Проект реализует две страницы:
- главная страница блога
- страница статьи

Дизайн основан на макете из Figma и выполнен с акцентом на семантическую вёрстку, адаптивность и структуру реальных frontend-проектов.

---

## 🧠 Цель проекта

Практика создания:
- семантической HTML-структуры
- адаптивной вёрстки
- CSS Grid и Flexbox
- методологии БЭМ
- базовой интерактивности (burger menu)

---

## 🛠 Технологии

- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid)
- JavaScript (burger menu)
- BEM methodology
- Responsive design
- Git / GitHub

---

## 📱 Адаптивность

Проект адаптирован под устройства:
- Desktop (1200px+)
- Tablet (768–1024px)
- Mobile (≤768px)



---

## ✨ Особенности реализации

- Семантические теги (`header`, `main`, `section`, `article`, `aside`, `footer`)
- БЭМ-нейминг классов
- CSS Grid для карточек блога
- Flexbox для элементов интерфейса
- Адаптивная сетка
- Бургер-меню для мобильной версии
- Доступность (alt, aria-label)

---

## 📌 Страницы

### Главная (Blog)
- список статей
- фильтры тегов
- адаптивная сетка карточек

### Статья (Article)
- текст статьи
- боковой блок с CTA
- адаптивная структура с `aside`

---

## 🚀 Запуск проекта

Открыть `index.html` в браузере или использовать Live Server.
